# TurnLight (ESP32 Super Mini + WS2812)

Кастомный задний модуль на адресных светодиодах:
- габарит (всегда включен)
- стоп (GPIO5, вход 3.3В после делителя с 12В)
- поворот (GPIO6, вход 3.3В после делителя с 12В)
- WS2812 data: GPIO4

## LED layout
3 концентрических кольца:
- Small: 7 LED (Ø25)
- Mid: 8 LED (Ø35) — поворотная анимация
- Large: 12 LED (Ø50)

Итого: 27 LED.

## Поведение
- Tail: все кольца красным, но с разной яркостью для объёма.
- Brake: малое + среднее + большое кольцо ярко-красным.
- Turn: волна по кольцам (малое → среднее → большое) с паузой в конце цикла, красный приглушён вдвое.
- Brake + Turn: активное кольцо волны оранжевое, остальные — как brake.
- Startup: 5 секунд дыхание синим по трём кольцам, затем белая вспышка и плавный переход в габариты.

## Настройки
Смотри `include/Config.h`:
- яркости tail/brake
- параметры стартового дыхания
- скорость/пауза волны поворота и задержка выключения
- глобальная яркость в режиме только tail

## Тестирование через Serial
Можно принудительно включать режимы из Serial Monitor (115200):
- `mode auto` — обычная работа от входов
- `mode tail` — только габарит
- `mode brake` — габарит + стоп
- `mode turn` — габарит + поворот
- `mode all` — все слои
- `mode test` — тест: стоп + поворот
- `mode off` — все выключено
