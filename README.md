# TurnLight (ESP32 Super Mini + WS2812)

Кастомный задний модуль на адресных светодиодах:
- габарит (всегда включен)
- стоп (GPIO5, вход 3.3В после делителя с 12В)
- поворот (GPIO6, вход 3.3В после делителя с 12В)
- WS2812 data: GPIO4

## LED layout
3 концентрических кольца:
- Small: 7 LED (Ø25)
- Mid: 8 LED (Ø35) — поворотная анимация
- Large: 12 LED (Ø50)

Итого: 27 LED.

## Поведение
- Tail: все кольца красным, но с разной яркостью для объёма.
- Brake: малое + большое кольцо ярко-красным.
- Turn: среднее кольцо оранжевым бегущим сегментом + мигание ~1.5 Гц.
- Brake + Turn: среднее = turn-анимация, малое+большое = brake.

## Настройки
Смотри `include/Config.h`:
- скорость/направление/старт поворотной анимации
- частота мигания
- яркости tail/brake
